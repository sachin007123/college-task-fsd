What is Git?

Git is a free, open-source distributed version control system. It tracks changes in your code over time, lets multiple people collaborate on the same project, and allows you to revert to previous versions if something goes wrong.

GitHub is a cloud-based platform that hosts Git repositories. Think of Git as the tool and GitHub as the online service that stores your work and makes collaboration easy.

🔀

Track Changes

Every change you make is saved as a "commit" — a snapshot of your project at that moment in time.

🤝

Collaborate

Multiple developers can work on the same project simultaneously without overwriting each other's work.

🌿

Branching

Create separate branches to build features or fix bugs without affecting the main codebase.

☁️

Backup & Restore

Your code is safely stored on GitHub. You can restore any previous version whenever you need it.
